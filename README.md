# Slagalica
 This game is playing on matrix field four by four. Ten fields of matrix have characters  A,B,C,D,E,F,G,H,I,J,K,L,M,N, and two field are free (white fields). In game user can move one freely selected field. Movement replaces free field with upper, lower, right or left character field. The object of this game is to move A field on target position.
